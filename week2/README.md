
# 📂 Week 2: Advanced Built-In Functions & Conditional State Analytics

Welcome to Week 2 of the Databricks training program! This repository contains the complete schema implementations, data seeding scripts, and production-ready SQL solution queries for the assignment.

The focus of this week is moving beyond basic queries to master specialized database functions, row-by-row math calculations, clean text formatting, time-interval precision tracking, and conditional classification metrics.

---

## 📅 Core Technical Focus Area
* **String Processing & Formatting:** Restructuring text casing (`INITCAP`, `UPPER`, `LOWER`) and extracting domain identifiers via substring parsing.
* **Defensive Numerical Mathematics:** Working safely with values using null handling (`COALESCE`), accurate rounding techniques (`ROUND`, `TRUNCATE`), and distance valuations (`ABS`).
* **Chronological Time & Date Metrics:** Extracting date segments, computing timestamp variations (`DATEDIFF`), and constructing sliding schedules relative to the current live runtime clock.
* **Conditional Control Pipelines:** Injecting advanced business rules directly into database relations via branching `CASE WHEN` control structures.
